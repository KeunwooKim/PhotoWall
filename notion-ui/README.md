# PhotoWall UI — Notion용 HTML

각 HTML 파일을 브라우저에서 열고 **전체 선택(Cmd+A) → 복사 → Notion 페이지에 붙여넣기** 하면 UI 목업이 들어갑니다.

## 파일 목록

| 파일 | 화면 | 경로 |
|------|------|------|
| `01-home.html` | 홈 (랜딩) | `/` |
| `02-profile.html` | 내 정보 | `/profile` |
| `03-settings.html` | 설정 | `/settings` |
| `04-wall-edit.html` | 개인 벽 편집 | `/wall/edit` |
| `05-wall-view.html` | 벽 보기 (공유 링크) | `/wall/[id]` |
| `06-shared-wall.html` | 공동 벽 편집 | `/shared/[id]` |
| `07-friends-panel.html` | 친구 패널 | 슬라이드 오버레이 |
| `08-shared-walls-panel.html` | 공동 벽 패널 | 슬라이드 오버레이 |
| `09-toolbar.html` | 꾸미기 메뉴 | 편집기 좌측 드로어 |
| `10-social-panel.html` | 응원 & 방명록 | 벽 보기 하단 시트 |

## 디자인 토큰

- 배경 `#fafafa` · 표면 `#ffffff` · 글자 `#1a1a1a`
- 보조 글자 `#737373` · 액센트 `#e8a0bf` · 액센트 진 `#c97b9a`
- 공동 벽 핑크 바 `#fce7f3` / `#fbcfe8`
