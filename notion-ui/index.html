PhotoWall UI 목업

Notion 문서용 HTML 파일 모음입니다. 각 페이지를 브라우저에서 열고 전체 복사하여 Notion에 붙여넣으세요.

파일 화면 경로
01-home.html홈 (랜딩)/
02-profile.html내 정보/profile
03-settings.html설정/settings
04-wall-edit.html개인 벽 편집/wall/edit
05-wall-view.html벽 보기 (공유)/wall/[id]
06-shared-wall.html공동 벽 편집/shared/[id]
07-friends-panel.html친구 패널오버레이
08-shared-walls-panel.html공동 벽 패널오버레이
09-toolbar.html꾸미기 메뉴드로어
10-social-panel.html응원 & 방명록바텀 시트
11-wall-private.html비공개 벽/wall/[id]
붙여넣기 방법
  1. 원하는 HTML 파일을 Chrome/Safari에서 엽니다
  2. 휴대폰 프레임 영역을 드래그하여 선택 (Cmd+A 전체 선택 가능)
  3. Notion 페이지에 Cmd+V로 붙여넣기
  4. 스타일이 일부 빠지면 /code 블록에 HTML 소스를 직접 붙여넣기
